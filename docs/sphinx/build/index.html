QEDlib is a Python library for expressively defining distributed hybrid classical / quantum computing tasks in a cloud environment.

It is a part of an open-source software architecture for hybrid quantum computing. It is the heart of a quantum cloud native programming and orchestration platform, implemented in Python, Rust and more.
